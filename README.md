# Sales Analysis Dashboard

This project presents an interactive Excel-based sales analysis dashboard developed using advanced data analysis features.

## Overview

The dashboard provides a comprehensive view of sales performance across multiple dimensions, enabling effective data-driven decision-making.

## Features

- Sales analysis by product category  
- Regional sales comparison  
- Monthly sales trend visualization  
- Payment method distribution analysis  
- Interactive filtering using slicers  

## Tools and Technologies

- Microsoft Excel  
- Pivot Tables and Pivot Charts  
- Slicers and Data Visualization Techniques
