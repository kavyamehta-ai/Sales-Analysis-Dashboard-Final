#  Sales Analysis Dashboard

This project is an interactive Excel dashboard created using:

- Pivot Tables
- Pivot Charts
- Slicers
- KPI Indicators

## Features

- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Payment Method Distribution
- Dynamic filtering using slicers

##  Tools Used

- Microsoft Excel
